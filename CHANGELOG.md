# Change Log

All notable changes to the "farzetki-darkblue-theme" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.2] - 2026-04-17
### Added
- **Multi-Theme Pack Support**: Expanded the extension to include 3 brand new, distinct themes:
  - **MatrixGreen**: A true dark-green hacker aesthetic with bright neon lime accents.
  - **BloodMoon**: A crimson/vampiric dark theme with intense red highlights.
  - **Unicorn**: A hyper-vibrant deep purple theme with hot pink and cyan magic.
- **Advanced UI Overhaul**: All 4 themes (including the original DarkBlue) now feature fully customized advanced VS Code interface elements, including restyled buttons, themed terminals, transparent scrollbars, active line highlighting, and color-coded Git integration.
- **Token Color (Syntax) Revamp**: Rewrote the entire `tokenColors` syntax highlighting dictionary so that code variables, strings, functions, numbers, and comments precisely match the aesthetic identity of each specific theme.

## [0.0.1] - 2026-04-15
### Added
- Initial release featuring the highly readable `Farzetki DarkBlue` theme.
