# Farzetki Theme Pack 🎨✨

Welcome to the **Farzetki Theme Pack**, a collection of highly curated, ultra-premium VS Code themes designed to match your mood, aesthetic, and productivity needs.

Whether you're hacking away late at night, feeling vampiric, or just want some vibrant neon magic, we've got you covered!

---

## 🌟 The Collection

This extension pack includes four meticulously crafted themes, each with fully custom UI elements (scrollbars, Git statuses, terminal colors) and perfectly matched syntax highlighting.

### 🔷 Farzetki DarkBlue
The original. A sleek, highly readable icy-blue aesthetic built to reduce eye strain during long coding sessions. It features deep navy backgrounds combined with cyan and arctic blue syntax highlights. Perfect for total focus.

### 🟩 Farzetki MatrixGreen
**Wake up, Neo.** 
A true hacker aesthetic. Featuring deep, dark pine backgrounds with intense neon lime accents. Variables, functions, and keywords glow beautifully. Feel like a cyberspace wizard every time you type.

### 🩸 Farzetki BloodMoon
For those who code in the dark. A crimson/vampiric dark theme utilizing near-black maroon backgrounds with aggressive red and gold highlights. It's intense, it's premium, and it makes every function feel powerful.

### 🦄 Farzetki Unicorn
Bring some magic to your IDE! A dark, cozy theme drenched in soft pink tones. Featuring deep rose-tinted backgrounds with hot pink and deep pink accents across the entire UI. Pure pink energy — no cyan, no distractions. For when you want your editor to feel warm and vibrant.

---

## 🛠️ Features

* **Complete UI Makeover:** Customized terminals, dropdowns, activity bar badges, and beautifully styled transparent scrollbar sliders.
* **Smart Syntax Highlighting:** Every language is beautifully colored to match your specific theme's identity. No more clashing colors.
* **Integrated Git Status:** Track your modified, untracked, and deleted files with colors that naturally match your theme's palette.

## 🚀 How to Use

1. Install the extension from the Marketplace.
2. Open the command palette (`Ctrl/Cmd + K` followed by `Ctrl/Cmd + T` or `Preferences: Color Theme`).
3. Select your favorite **Farzetki** theme from the list!
4. Switch themes anytime based on your mood.
